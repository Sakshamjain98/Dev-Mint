How to Use the Calculator

An Arithmetic-Geometric Progression (AGP) is a sequence where each term after the first is found by:

  1. Adding a constant (arithmetic part)
  2. Multiplying by a constant (geometric part)

Input Parameters:

  • First Term (a): The initial value of the sequence
  • Common Difference (d): The arithmetic increment
  • Common Ratio (r): The geometric multiplier
  • Number of Terms (n): How many terms to calculate
