Frontend Layouts Cheat Sheet

Key CSS Layout Techniques

  • Flexbox: Align and distribute items along a single axis.
  • Grid: Create complex, two-dimensional layouts with ease.
  • Positioning: Use absolute, relative, fixed, and sticky for custom placement.
  • Float: Wrap text or elements around floated items (deprecated for layout purposes).

Common Layout Patterns

Pattern Description CSS Example
Two-Column Layout Sidebar and Main Content. 
.container {
  display: flex;
}
.sidebar {
  width: 25%;
}
.main-content {
  flex: 1;
}
Holy Grail Layout Header, Footer, Sidebar, Main Content. 
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-rows: auto 1fr auto;
  height: 100vh;
}
Centering Content Horizontally and vertically align an element. 
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
Responsive Layout Make layouts adapt to screen sizes. 
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}

Best Practices

  • Use rem and em for scalable spacing and font sizes.
  • Set box-sizing: border-box; globally for consistent sizing.
  • Leverage CSS variables for theme consistency.
  • Combine Grid and Flexbox for advanced layouts.
  • Test layouts on different devices and browsers.
